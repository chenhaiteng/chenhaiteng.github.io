使用適用於 Mac OS X 的 iWeb、Pages 和其他許多應用程式快速且輕鬆地製作並發佈您自己的網站。

使用 Mac 製作並發佈您自己的網站簡直是輕而易舉。當您的網站準備就緒時，發佈網站也是一樣容易。

打開“系統偏好設定”並按“共享”一下，然後選擇“網頁共享”。

這樣就大功告成了。現在您可以在住家或公司的專用網路上連接網站。

如果您已連接 Internet，世界各地的朋友也可以連接您的網站。只要將“共享”偏好設定中顯示的位址傳送給他們即可。

功能強大的 Apache

“網頁共享”功能是在 Apache 網頁伺服器上建立的，其為符合業界標準的技術，隨附於 Mac OS X。如需更多 Apache 網頁伺服器的相關資訊，請參閱 「Apache 手冊」。
